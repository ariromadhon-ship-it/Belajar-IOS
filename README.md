## Getting Started

### Prerequisites
- Xcode 14.0 or later
- iOS 15.0 or later
- Swift 5.7 or later

### Building
1. Navigate to project folder
2. Open `.xcodeproj` in Xcode
3. Select simulator or device
4. Press Cmd+B to build
5. Press Cmd+R to run
